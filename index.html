Projeto 1

Esse projeto o objetivo era criar um layout HTML explorando as cores no CSS, conforme exemplo fornecido pela professora, usei os níveis de título H1 a h6, os elementos de margin de 80 PX, padding 80px, border 20 PX. Uma lista não ordenada e uma lista ordenada contendo alguns links que direcionava para uma página especifica, meu maior desafio nesse projeto foi escolher a cor certa,sem ideia do que estava fazendo, escolhi cores muito escuras, depois que entreguei vi que estava horroroso, com auxilio da professora fiz a correção na aula seguinte, e melhorei o projeto.

projeto 2

Projeto 6

Com utlização do JavaScript o objetivo desse projeto era criar um programa que solicitasse ao usuário um número inteiro e verificava se esse número atende aos seguintes critérios: É um número par. É um número positivo (maior que zero). É um número menor que 100. O programa deve imprimir a mensagem "O número atende aos critérios.", se o número atender a todos os critérios acima, caso contrário, deve imprimir a mensagem "O número não atende aos critérios. além do solicitado pela professora, criei um programa que também solicitava nome do usuário e o desejava boas vindas, também inclui uma imagem de fundo usando html e CSS.

Projeto 7

Criei uma calculadora para realizar o cálculo de 2 números com botões somar, diminuir, multiplicar, dividir e limpar. ao clicar em um dos botoes deve imprimir a mensagem: A soma, a subtração,multiplcação ou divsão é: ao chamar o botão limpar retorna para mensagem 'resultado'.Para o fundo, apliquei background-color deixando o projeto mais colorido.

Para esse projeto, criei arquivo com estrutura básica HTM, um menu, uma logo, alterei icone do html, inclui uma imagem de fundo responsiva, um footer com icones de rede sociais, para personalizar o site, usei arquivo CSS apliquei flex box deixando os ícons na vertical quando tela grande, e horizontal quando tela pequena, para isso foi aplicado @ midea com interrupçao em 48em..
